# Email Validation Tool

This is a simple Python script for validating a list of email addresses using the Microsoft Azure AD email validation API. It checks each email address and saves the valid ones to a given file name.

## Prerequisites

Before you can use this script, make sure you have the following prerequisites installed:

- Python 3.x
- requests library (you can install it using `pip install requests`)
- colorama library (you can install it using `pip install colorama`)

## Usage

1. Clone this repository or download the `email_validation.py` script.

2. Run the script by executing the following command in your terminal:

   ```bash
   pip install -r requirements.txt
   python o365.py 
   ```
